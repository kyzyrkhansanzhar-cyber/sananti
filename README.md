Sananti: Go Deception-Defense & Anti-Fraud Security Library

[English]
Sananti is a cybersecurity library written in Go. It provides active deception defense, IP rate limiting, webhook alerts, and transaction anti-fraud scanning.
What it does:
- Automatically blocks malicious bot IP addresses using honeypots and decoy routes.
- Restricts excessive request rates using a thread-safe Token Bucket Rate Limiter.
- Dispatches real-time security alerts to external APIs/webhooks (e.g. Telegram/Slack).
- Evaluates risk factors for credit card payments in real-time.
How to install:
go get github.com/kyzyrkhansanzhar-cyber/sananti

Sananti - это библиотека безопасности на языке Go. Она обеспечивает активную защиту от ботов, ограничение частоты запросов (rate limiting), отправку уведомлений через вебхуки и сканирование платежей на наличие фрода.

- Автоматически блокирует IP-адреса вредоносных сканеров и ботов с помощью ловушек.

- Отправляет мгновенные уведомления о критических угрозах на внешние вебхуки (например, Telegram/Slack).
- Оценивает уровень риска платежных транзакций в реальном времени.

go get github.com/kyzyrkhansanzhar-cyber/sananti
