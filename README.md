Sananti: Go Deception-Defense & Anti-Fraud Security Library

[English]
Sananti is a cybersecurity library written in Go. It provides active deception defense and transaction anti-fraud scanning. It automatically blocks malicious bot IP addresses using honeypots and decoy routes, and evaluates risk factors for credit card payments in real-time.
How to install:
go get github.com/kyzyrkhansanzhar-cyber/sananti
